# news-agent
A modern Streamlit web application that fetches and displays real-time news using APIs, with a clean UI and dynamic content rendering.A smart and lightweight News Fetcher Agent that automatically retrieves the latest news articles from various sources using APIs and displays them in a clean and structured format. This project is designed to help users stay updated with real-time news in an efficient and user-friendly way.

🚀 Features
🔎 Fetch latest news from APIs
🌐 Supports multiple news categories (Technology, Sports, Business, etc.)
⚡ Fast and efficient data retrieval
🖥️ Simple and interactive UI (Streamlit-based if used)
🔄 Real-time updates
📄 Clean and readable news display
